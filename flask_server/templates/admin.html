Chatbot Overview

Welcome to the Admin Panel for the AI-powered chatbot. This chatbot is designed to assist users by providing relevant information and answering queries efficiently.

Our chatbot is built using Natural Language Processing (NLP) techniques with frameworks such as NLTK and spaCy. It leverages a trained neural network model to understand user intent and provide meaningful responses.

The system is integrated with a Flask-based backend and supports dynamic learning, ensuring continuous improvements in response accuracy over time.
